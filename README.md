# STRATA — Trading Utilities Desk

An interactive, single-file web app for working out trading math before you place a trade — 
built for both stocks and forex.

## Features
- **Position Size Calculator** — sizes a trade from your account balance, risk %, entry, and stop
- **Risk/Reward Mapper** — visualizes risk vs. reward distance and the win rate needed to break even
- **P&L Calculator** — long/short, leverage, margin, and fees for stocks or forex
- **Strategy Library** — reference cards for trend following, breakout, mean reversion, scalping, 
  swing, and range trading
- **Equity Curve Simulator** — Monte Carlo–style simulation showing what a given win rate and 
  avg win/loss actually looks like over many trades

## Usage
Open `index.html` in any browser — no build step, no dependencies, no server required.
